# Kerala Ministry Website

This project was developed as part of a Day 2 internship assignment using HTML, CSS, and Bootstrap. The goal was to learn how Bootstrap can be integrated into a website, use its pre-built components, customize them with CSS, and create a structured multi-page website.

The website provides information about the Kerala Ministry through a Home page, Ministers page, individual Minister Profile pages, and a Contact page.

## Features

* Home page with Bootstrap Carousel
* Ministers page with profile cards
* Individual minister profile pages
* Contact page with form layout
* Navigation bar for page routing
* Custom styling using CSS
* Bootstrap components such as Navbar, Carousel, Cards, and Forms

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5

## What I Learned

* Integrating Bootstrap using CDN
* Working with Bootstrap components
* Customizing Bootstrap styles with CSS
* Creating and linking multiple pages
* Organizing project files and folders
* Designing a clean and consistent user interface

## Project Structure

```text
KERALA_MINISTRY/

├── STYLE.CSS
```

## Running the Project

Open `INDEX.HTML` in any web browser to view the website.
